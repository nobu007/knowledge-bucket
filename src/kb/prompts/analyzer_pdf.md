# PDF Analyzer

## 対象

PDF文書、ホワイトペーパー、技術レポート、マニュアル。

## 抽出対象

- タイトル
- 著者（わかる場合）
- テキスト内容
- 章構造（目次から）
- 図表キャプション（わかる場合）

## 分析のポイント

1. 文書の全体像を要約する
2. なぜこの文書が自分にとって重要かを書く
3. 最大5個の重要ポイントを抽出する
4. 文書の中心概念や手法をprimary_conceptsに挙げる
5. 関連する背景概念をcandidate_conceptsに挙げる
6. 言及されているツール、組織、規格をentitiesに挙げる

## 注意

- PDF全文はGitに保存しない。要約、メモ、短い引用に留める。
- Raw PDFはS3/R2等の外部ストレージに保存し、source URLで参照する。

## 入力形式

```
Title: <PDFタイトル>

<PDFから抽出したテキスト>
```
