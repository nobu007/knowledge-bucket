# Paper Analyzer

## 対象

学術論文、arXiv論文、カンファレンス論文。

## 抽出対象

- タイトル
- 著者リスト
- Abstract（要約）
- DOI（わかる場合）
- arXiv ID（わかる場合）
- 主要な結論と貢献
- 提案手法の核心

## 分析のポイント

1. 論文の目的と貢献を要約する
2. なぜこの論文が自分にとって重要かを書く
3. 最大5個の重要ポイント（手法の新規性、実験結果、限界など）を抽出する
4. 論文が提案または改善する手法や概念をprimary_conceptsに挙げる
5. 関連手法やベースライン技術をcandidate_conceptsに挙げる
6. 使用データセット、評価指標、ツール、著者の所属組織をentitiesに挙げる

## 入力形式

```
Title: <論文タイトル>
Authors: <著者>
DOI: <doi>
arXiv: <arxiv_id>

<Abstract + 本文抜粋>
```
