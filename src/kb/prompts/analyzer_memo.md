# Memo Analyzer

## 対象

個人のメモ、アイデア、思考の断片、議事録。

## 抽出対象

- メモの内容
- 作成日時（front matterから）
- ユーザー指定タグ（あれば）

## 分析のポイント

1. メモの核心を簡潔に要約する
2. なぜこのメモを残したかを推測して書く
3. 最大3個の重要ポイントを抽出する（メモが短い場合は少なくてよい）
4. メモの中心概念をprimary_conceptsに挙げる（最大2個でよい）
5. 関連しそうな概念をcandidate_conceptsに挙げる
6. confidenceは控えめに設定する（メモは情報が少ないため）

## メモ特有の注意

- メモは情報が不完全なことが多い。推測部分は明示する。
- display_tagsは最大3個でよい。
- entitiesは明確に言及されているものだけ挙げる。
- confidenceは0.3〜0.6程度に設定する。
- importanceも控えめに0.1〜0.4程度にする。

## 入力形式

```
Title: <メモタイトル（なければ先頭行）>

<メモ本文>
```
